# 🌙 Dream Diary (드림다이어리)

A cozy desktop journaling app — built solo, with Claude as my co-developer.
클로드를 공동 개발자 삼아 혼자 만든, 아늑한 데스크톱 다이어리 앱.

> Made by **SOMBI** — an ordinary Korean creator who builds the tools he wishes existed.
> 평범한 한국 창작자가 "있으면 좋겠다" 싶은 도구를 직접 만듭니다.

---

## ✨ What is it? / 무엇인가요?

Dream Diary is a desktop diary app (built with **Tauri**), focused on making the diary truly *yours* through presets, skins, and fonts.

드림다이어리는 **Tauri**로 만든 데스크톱 다이어리 앱입니다. 프리셋·스킨·폰트를 자유롭게 바꿔 **나만의 다이어리**로 꾸밀 수 있어요.

## 🎨 Features / 기능

- 📔 **Personal journaling** with a calm, clean interface / 차분하고 깔끔한 개인 일기
- 🎨 **Skins & themes** — skin packs that change the whole look / 전체 분위기를 바꾸는 스킨팩
- 🔤 **Custom fonts** / 다양한 폰트 지원
- 🧩 **Presets** — ready-made setups applied in one click / 원클릭 프리셋
- 🖼️ **Built-in image tools** — e.g. permanently embedding images into shared HTML / 내장 이미지 도구 (공유 HTML에 이미지 영구 내장 등)

## 🛠️ Part of a family of tools / 함께 만든 도구들

I build small desktop tools for the creative community — all solo, with Claude:
창작 커뮤니티를 위한 작은 도구들을 클로드와 함께 혼자 만들고 있어요:

- **NAI Lab / NAI Studio** — tools for NovelAI image creation / NovelAI 이미지 생성 도구
- **MessengerGod** — an Electron port of an open-source RisuAI chat plugin / 오픈소스 RisuAI 플러그인 Electron 포팅
- **Cursy** — a mouse-cursor maker / 마우스 커서 제작 앱

## 💜 About / 소개

I'm not a professional developer — just an ordinary person who loves turning ideas into real, working things. Claude is my co-developer, my translator, and the reason I can build and share at all.

저는 전문 개발자가 아니라, 상상한 걸 실제로 만들어보는 게 좋은 평범한 사람입니다. 클로드가 제 공동 개발자이자 번역가이며, 제가 무언가를 만들고 나눌 수 있는 이유입니다.
